# Team Practice Project. 

## Description
A hands-on group project designed by instructors to:
- Practice real-world React development using concepts learned in class.
- Simulate team collaboration with Git, code reviews, and task delegation.
- Emulate agile workflows (PRs) in a safe learning environment.

## Key Goals 
- A platform to practice what you've been learning so far
- Work on branching, merging, and conflict resolution
- Experience code reviews and documentation
- Present work as a team

## Installation

1. Fork the repository 
Creates a copy on GitHub under your account using the fork button at the top right corner of this page. 

2. Clone the repository

```bash
git clone https://github.com/your-github-username/project-name.git
```

3. Navigate to the project directory
```bash
cd your project
```

4. Create a branch you'll like to work on and checkout to the branch

```bash 
git checkout -b branch-name
```

5. Install all dependencies

```bash 
npm install
```


## Usage
Run the following command to start the project server.

```bash
npm run dev
```
Open the browser and go to http://localhost:5173 to view the project.


## Contribution Guidelines

1. PR Review Process
    - All pull requests must be reviewed and approved before merging to maintain code quality and prevent merge conflicts.

2. Collaborator Access
    - Contributors will be added as collaborators to the repository after their first PR is successfully merged.

3. Recognition
    - Only accepted and merged PRs will count toward official contributions

4. Active Contributor Policy
    - To maintain project quality, only users with approved and merged pull requests will be recognized as active project members.
